# Support

ChurchCore Academy is an early-stage open-source project and does not currently provide commercial support or guaranteed response times.

## Where to Ask

- Use a GitHub issue for reproducible bugs.
- Use a feature request for product proposals and workflow gaps.
- Use pull requests for focused, verified changes.
- Use private vulnerability reporting for security concerns.

Before opening an issue:

1. read the [README](README.md) and [documentation index](docs/README.md)
2. search existing issues
3. reproduce against the latest `main`
4. run the relevant tests and include the failing command

## Useful Diagnostic Information

Include:

- operating system
- Node.js and npm versions
- Supabase CLI version
- current commit
- route or command involved
- sanitized logs and error output
- expected and observed behavior

Never attach `.env.local`, access tokens, service-role keys, database credentials, student records, or other private data.
